# これは何?

バーチャル背景ジェネレーター
https://backscreen-generator.web.app/#/

Zoomでの懇親会などで、バーチャル背景に自己紹介を記述したい場合に
簡単に画像を作りたいという課題を解決します。

上記URLにアクセスし、表示したい項目を入れると、ボタンひとつでPNG画像をダウンロードできます。

## Architecture / アーキテクチャ
* Flutter Web + Firebase Hosting
